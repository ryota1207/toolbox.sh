# toolbox.sh
 
「toolbox.sh」Linuxコマンドでの作業を効率化させるエンジニア向けツールボックスです。

### ■「toolbox.sh」の特徴

1. 簡単インストール<br>
2. スマート拡張設計
3. 日本語ドキュメント
